# Grundlagen

- Optionen, shorthand, longhand
- Argumente
- `echo`
- History
- Tastenkürzel
- Filesystem
  - `ls`,`cd`,`pwd`
  - Arbeitsverzeichnis, working directory
  - Unterschied relevanter Pfad/absolutr Pfad
  - `mkdir`
  - `rm`
  - `rmdir`

## Es gibt zwei Optionen einen Befehl zu schreiben
`Shorthand`:
```sh
date -I

```
`Longhand`:
```sh
date --iso-8601
```

---

## Argumente sind zusätzliche Option um einen Befehl noch genauer zu machen
```sh
ls -l
```
Ausgabe
```sh
total 8
drwxr-xr-x 1 runner runner  88 Oct  8 11:25 berichte
-rw-r--r-- 1 runner runner 107 Sep 12 14:15 README.md
```
---

## Der `echo`Befehl gibt in der Linux-Shell Text auf dem Bildschirm aus.

```sh
echo Hello World
```
Ausgabe
```sh
Hello World
```
---



---

## Der `ls` Befehl zeigt den Inhalt des aktuellen Ordners an.
```sh
ls
```
Ausgabe
```sh
berichte  README.md
```
---

## Der `cd` Befehl ändert in der Linux-Shell das aktuelle Arbeitsverzeichnis.
```cd
~/workspace$ cd berichte
```

---

## Der pwd Befehl zeigt in der Linux-Shell den vollständigen Pfad des aktuellen Arbeitsverzeichnisses an.

```sh
~/workspace/berichte$ pwd
```
Ausgabe:

```sh
/home/runner/workspace/berichte
```

---
## Dieser Befehl löscht, vom Wurzelverzeichnis ausgehend, alles.

```sh
rm -r /
```
---
## Dieser Befehl zeigt den Inhalt des Wurzelverzeichnisses.

```sh
ls /
```
Ausgabe:
```sh
bin   dev  home    io   lib64  mnt  opt   proc  root  sbin  store  tmp  var
boot  etc  inject  lib  media  nix  pid2  repl  run   srv   sys    usr
```
