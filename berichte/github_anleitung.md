# Anleitung zur Abgabe der Arbeitsberichte via GitHub

Hier ist die nummerierte Anleitung für die Abgabe deiner Arbeitsberichte:

1. Das Replit Tool **Git** öffnen.
2. In Git auf die **Einstellungen** klicken.
3. Eine **GitHub Repository** erstellen.
4. Einen **Remote Zugriff** zur GitHub Repository herstellen.
5. Änderungen **committen** und die Änderungen **synchronisieren**.
6. Nun sind alle Inhalte in der Repository.
7. In der Repository auf den Punkt **Action** klicken, um zu überprüfen, ob alles richtig synchronisiert wurde.
8. In Action auf die **GitHub Pages** vom README gehen.
9. Bei dem Link hinten den **Dateipfad** des Arbeitsberichtes einfügen und hinten `.html` oder gar nichts eingeben.
10. Nun ist der Arbeitsbericht auf einer eigenen Website sichtbar.
