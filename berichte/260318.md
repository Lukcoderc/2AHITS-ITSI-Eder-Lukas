## Passwort Check manipulieren

Datei herunterladen:
```sh
curl -O https://www.franzmatejka.at/htl/doc/ITSI_2_linux/opcode_patching/bin/replit/pwcheck
```
Programm ausführbar machen:
```sh
chmod u+x pwcheck
```

Binärcode ansehen:
```sh
xxd pwcheck
```

GNU Debugger Installieren:
```sh
gdb pwcheck
```

Programm mit dem Debugger öffnen:
```sh
gdb pwcheck
```

Breakpoint bei Main machen:
```sh
b main
```

Programm Starten
```sh
run
```

Mit `x/5i $pc` und `ni` zur Speicheradresse des `je` Befehls nach dem Check Passwort printf navigieren

Gesamten Befehl auf 0 setzen Damit die Passwortüberprüfung übersprungen wird.

Programm starten und irgendetwas in die Passwortabfrage eingeben.

Passwortabfrage wird übersprungen und das Passwort wird angezeigt.
