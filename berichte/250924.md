# Grundlagen der Linux Shell

Shell = textorienterte Schnittstelle zu einem Betriebssystem (Kernel)

Anzeige in der Konsole: Prompt
Eingabe nach dem Promt

Beispiel: der Befehl `date`

```sh
$ date
```

Ausgabe:

```
Wed Sep 24 07:11:47 AM UTC 2025
```

Befehle können Oprionen haben, diese werden mir `-` angegeben.

z.B.

```sh
$ date -I
```

Ausgabe:

```
2025-09-24
```

Optionen aus einem Buchstaben = shorthand

longhand mit 2x `--`:

```sh
date --iso-8601
```

Mehr Informationen zu einem Befehl:

-KI
-häufig gibt es die Option `--help`

```sh
$ date --help
```

-man pages

```sh
$ man date
```
