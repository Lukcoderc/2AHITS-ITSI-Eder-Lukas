## Neue Methode der Abgabe von Arbeitsberichtet

1. Das Replit Tool Git öffne ![image](img/gitHub_x_Replit_1.jpeg)
2. In Git auf die Einstellungen klicken ![image](img/gitHub_x_Replit_2.jpeg)
4. Eine Github Repository erstellen ![image](img/gitHub_x_Replit_3.jpeg)
5. Einen Remote Zugriff zur Github Repository herstellen ![image](img/gitHub_x_Replit_4.jpeg)
6. Änderrungen commiten und die Änderungen synchronisieren ![image](img/gitHub_x_Replit_5.jpeg)
7. Nun sind alle Inhalte in der Repository
8. In der Repository auf den Punkt Action klicken um zu überprüfen ob alles richtig synchronisiert ist![image](img/gitHub_x_Replit_6.jpeg)
9. Im README einen link zu dem Aktuellen Arbeitsbericht einfügen
10. In Action auf den neusten commit gehen und dort die Git-Hub pages öffnen ![image](img/gitHub_x_Replit_7.jpeg)
11. Bei dem Link hinten den Dateipfad des Arbeitsberichtes einfügen und hinten `.html` oder gar nichts eingeben
12. Nun ist der Arbeitsbericht auf einer eigenen Website Sichtbar
