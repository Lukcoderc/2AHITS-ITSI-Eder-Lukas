## Neue Methode der Abgabe von Arbeitsberichtet

- Das Replit Tool Git öffnen
- In Git auf die Einstellungen klicken
- Eine Github Repository erstellen
- Einen Remote Zugriff zur Github Repository herstellen
- Änderrungen commiten und die Änderungen synchronisieren
- Nun sind alle Inhalte in der Repository
- In der Repository auf den Punkt Action klicken um zu überprüfen ob alles richtig synchronisiert hat
- In Action auf die Github Pages von dem Read ME gehen
- Bei dem Link hinten den Dateipfad des Arbeitsberichtes einfügen und hinten `.html` oder gar nichts eingeben
- Nun ist der Arbeitsbericht auf einer eigenen Website Sichtbar
