
Hinweis: Die -I Option zeigt Default nur das Datum an.

## Übung (Befehle – Bewegen und Kopieren)

### Aufgabenstellung:

Informiere dich über die Befehle `rm`, `rmdir`, `cp` und `mv`. Probiere die Befehle aus und dokumentiere deine Erkenntnisse im Arbeitsbericht.

### Aufgabe:

### `rm` Befehl:

Der rm Befehl wird verwendet um eine Datei zu löschen.

Eingabe:
```sh
rm test.txt
```
Keine Ausgabe weil bei Erfolg eines Befehls wird meistens nicht ausgegeben.

### `rmdir` Befehl:

Der rmdir Befehl wird verwendet um ein leeres Verzeichnis zu löschen.

Eingabe:
```sh
rmdir test
```

Das gesamte Verzeichnis kann mit `rm -r` gelöscht werden.

Eingabe:
```sh
rmdir -r test
```

### `cp` Befehl:

Der cp Befehl wird verwendet um Verzeichnisse oder Dateien zu kopieren.

Eingabe:
```sh
cp test.txt ~/workspace/abcd
```
### `mv` Befehl:

Der mv Befehl wird verwendet um Verzeichnisse oder Dateien zu verschieben oder Dateien umzubenennen.

Eingabe:
```sh
mv test.txt ~/workspace/abcd
```

## Übung (Bewegen und Kopieren)

```sh
mv neu.txt unbenannt.txt
```


```sh
mkdir kopie_von_first

```
5.
```sh
rm -r first_dir
```

### Aufgabenstellung:

Informiere dich über die Optionen der ls, mv und rm Kommandos. Probiere aus und dokumentiere deine Erkenntnisse im Arbeitsbericht.

### Aufgabe:

`ls` Befehl:

Der Befehl dient dazu, den Inhalt eines Verzeichnisses (Ordners) aufzulisten.

Beispiel:
```sh
ls #Um den Inhalt des aktuellen Ordners aufzulisten
abcd berichte README.md #Ausgabe
```
oder
```sh
ls ~/workspace/berichte/ #Um den Inhalt eines bestimmten Ordners aufzulisten
250910.md  250924.md  251008.md  251022.md  _EDER_Lukas.md  img #Ausgabe
```

`mv` Befehl:

Der Befehl mv wird verwendet, um Dateien oder Verzeichnisse zu verschieben oder umzubenennen.

Beispiel:
Datei oder Ordner verschieben
```sh
mv quelle ziel
```
Datei umbenennen
```sh
mv alt.txt neu.txt
```

`rm` Befehl:

Der Befehl rm steht für remove und dient dazu, Dateien oder Verzeichnisse zu löschen. Gelöschte Dateien sind nicht wiederherstellbar.

Beispiele:
Eine Datei löschen:
```sh
rm dokument.txt
```
Mehrere Dateien löschen:
```sh
rm datei1.txt datei2.txt datei3.txt
```
Einen ganzen Ordner samt Inhalt löschen:
```sh
rm -r alter_ordner/
```
