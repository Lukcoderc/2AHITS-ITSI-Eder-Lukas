## Übung (Sort)

### Aufgabenstellung:
Mit der Datei shopping.txt – löse jeden Punkt durch einen Einzeiler mit Hilfe von Pipes:

- Sortiere in umgekehrter Reihenfolge (eine Option von sort)
- Sortiere nach der 2ten Spalte – Obstnamen (dies ist ebenfalls eine Option von sort)
- Nimm alles bis auf die ersten 5 und letzten 5 Zeilen, sortiere das nach Personennamen in absteigender Reihenfolge ('Z' zuerst). Sortiere das Ergebnis aufsteigend nach Obstnamen und gibt die letzten 3 Zeilen des Endergebnisses aus.

### Aufgabe

1. Sortieren in umgekehrter Reihenfolge:
```sh
cat shopping.txt | sort -r 
```

2. 
```sh
cat shopping.txt | sort -t ' ' -k2
```

3. 
Zuerst mit `tail -n +6` die ersten 5 Zeilen überspringen. `+6` weil er erst ab Zeile 6 beginnen soll. `head -n -5` entfernt die letzten 5 Zeilen. Sort sortiert es Rückwärts und mit `sort -t ' ' -k2` sortiert er nach dem Obst. Das letzte Tail gibt die letzten 3 Zeilen aus.
```sh
tail -n +6 shopping.txt | head -n -5 | sort -r | sort -t ' ' -k2| tail -n3
```
Ausgabe:
```sh
Mia strawberries 25
Victor tangerines 1
Lily tangerines 15
```

## Übung (Namensliste)

### Aufgabenstellung:
Erstelle mit cut, sort und uniq eine alphabetisch sortierte Liste an Namen die in shopping.txt vorkommen. Jeder Name soll in dieser Liste nur einmal stehen.

### Aufgabe:

- `cut -d ' ' -f1 shopping.txt` trennt jede Zeile in Felder. Jedes Feld wird an einem Lehrzeichen getrennt. Dadurch entstehen in jeder Zeile 3 Felder und mit `-f1` wird das erste Feld also der Name ausgeschnitten.
- `sort` sortiert alphabetisch
- mit `uniq` steht jedes Wort nur einmal da
```sh
cut -d' ' -f1 shopping.txt | sort | uniq
```

## Übung(Numeric Sort)

### Aufgabenstellung:
Recherchiere die sort Option -n, --numeric-sort und erkläre (auch anhand eines Beispiels) die Auswirkung wenn diese Option angegeben oder nicht angegeben ist.

Hinweis: Sortiere eine Datei die ein- und zweistellige Zahlen gemischt untereinander enthält.

### Aufgabe

`-n` und `--numeric-sort` sortiert nach Zahlen aufsteigend.
Ohne `-n`:
```sh
sort zahlen.txt
```
Ausgabe:
```sh
1
10
11
2
20
3
```

Mit:
```sh
sort -n zahlen.txt
```
Ausgabe:
```sh
1
2
3
10
11
20
```

## Übung (Sort Optionen)

### Aufgabenstellung
Erkläre – mit Beispielen – die sort Optionen:

- -m, --merge
- -R, --random-sort

### Aufgabe

`-m` und `--merge` führt mehrere bereits sortierte Dateien zusammen.

Beispiel:

```sh
a.txt
1
3
5
```
```sh
 b.txt
2
4
6
```
```sh
sort -m a.txt b.txt
```
Ausgabe:
```sh
1
2
3
4
5
6
```
`-R` und `--random-sort` Gibt die Zeilen in zufälliger Reihenfolge aus.

Beispiel:
```sh
sort -R zahlen.txt
```
