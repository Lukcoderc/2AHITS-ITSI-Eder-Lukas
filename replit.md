# replit.md

## Overview

This repository contains educational exercises and work reports for a 2AHITS ITSI (IT Systems and Infrastructure) class. The content focuses on learning Linux shell commands, Markdown documentation, and basic programming concepts. Students document their progress through work reports written in Markdown format, covering topics like file manipulation, stdin/stdout redirection, and shell utilities.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Documentation Structure
- **Work Reports**: Located in `berichte/` directory as Markdown files with date-based naming (YYMMDD.md format)
- **Sample Data Files**: Text files in root directory used for shell command exercises (zahlen.txt, shopping.txt, passwd_numbered.txt)
- **README**: Links to class homepage and published work reports

### Content Publishing
- Reports are written in Markdown and converted to HTML for web viewing
- GitHub Pages integration for publishing student work reports
- Git workflow used for version control and submission

### Exercise Types
The repository demonstrates:
1. **Shell Commands**: head, tail, cat, ls, mkdir, touch, rm, seq, nl, date
2. **I/O Redirection**: stdin/stdout manipulation with <, >, >>
3. **C/C++ Compilation**: Basic program compilation with g++
4. **Markdown**: Documentation formatting including code fences, links, images

## External Dependencies

### Publishing Platform
- **GitHub Pages**: Used for hosting rendered Markdown reports as HTML web pages
- **GitHub Actions**: Automated workflow for building and deploying documentation

### Development Environment
- **Replit**: Primary development and execution environment
- **Git**: Version control integrated through Replit's Git tool
- **GCC/G++**: C/C++ compiler for programming exercises

### No Database
This is a documentation-only repository with no database requirements.

### No External APIs
No third-party API integrations are used.
